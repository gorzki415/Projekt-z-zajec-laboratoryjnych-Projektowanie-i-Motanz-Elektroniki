# Projekt-z-zajec-laboratoryjnych-Projektowanie-i-Motaz-Elektroniki  
English below  
  
Mini projekt wykonany podczas zajęć laboratoryjnych z przedmiotu "Projektowanie i Montaż Elektroniki"  
  
Jest to mini projekt wykonany na zaliczenie z części laboratoryjnej przedmiotu "Projektowanie i Montaż Elektroniki".  
  
Z racji na wymóg prowadzącego projekt ten był wykonany w Autodesk Eagle.  
  
Projekt ten wykorzystuje dwa wzmacniacze TD7294 do wzmacniania otrzymywanego sygnału audio.  
  
Semestr 4  

English:  

This is a mini project created as part of the laboratory assessment for the course "Electronics Design and Assembly".  
  
Due to the instructor’s requirements, the project was designed using Autodesk Eagle.  
  
The project utilizes two TDA7294 amplifiers to amplify the incoming audio signal.
